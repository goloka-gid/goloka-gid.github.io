# goloka-gid.github.io
# ✨ Путешествие Элли: 30 дней чудес

Мини-приложение для Telegram с медитациями-играми.

### Структура проекта:
* `audio/` — аудиофайлы медитаций (формат: `dayX_story.mp3`, `dayX_child.mp3` и т.д.)
* `texts/` — JSON файлы с текстами историй и практик.
* `index.html` — основной код приложения.
